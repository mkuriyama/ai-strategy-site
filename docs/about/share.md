# このプロジェクトを紹介する

本プロジェクトにご関心がありそうな方へお渡しいただける、**A4の紹介チラシ（印刷用）**を
用意しています。社内での共有や、知人へのご紹介にお使いください。

- A4・**カラー印刷でも白黒印刷でも**読みやすいデザイン
- **片面**だけでも概要が伝わり、**両面**でより詳しくなる構成
- 開催日程など最新情報を反映し、**回ごとに更新**しています

[紹介チラシを開く（印刷用・別タブ）](../assets/flyer.html){ .md-button .md-button--primary target=_blank }

開いたページからブラウザの印刷（Windows: `Ctrl + P` ／ Mac: `⌘ + P`）でA4印刷できます。
Webで紹介する場合は、本サイトのURL **<https://ai-strategy.antecanis.com/>** をそのまま
お伝えください。
