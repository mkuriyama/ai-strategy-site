# このプロジェクトが「何でないか」

本プロジェクトは、以下とは異なる場として設計しています。似た場はすでに数多く存在し、
本プロジェクトはそれらを代替するものではありません。

**個別ツールの勉強会**（Claude CodeやOpenClawの導入講座など）
:   有償・無償ともに数多く見つかります。

**体系的な学びの場**（AIや戦略の理論面の学習・体系化）
:   経営大学院などで提供されています。主催の栗山が教員を務めるグロービス経営大学院・
    多摩大学大学院でも各種提供しています。

**活用事例共有会**（○○業界でこんな導入、こんな効果）
:   AI関連のイベント・カンファレンス等で多数開催されています。

**大企業向けのAIシステム実装、個別企業でのAI導入相談**
:   AIベンダー各社が対応しています。栗山も参画するVISITS Technologies社でもお受けします。

---

本プロジェクトが焦点を当てるのは、これらの「あいだ」にある領域です。すなわち、
**事業戦略の問いから出発し、自ら手を動かしてAIで動く仕組みを作り、その戦略インパクトを
体感する**——という、理論でも単発ツールでも事例共有でもない実践の場です。

設計の詳細は **[設計思想](philosophy.md)** を、対象とする方は
**[参加する](../join/index.md)** をご覧ください。
