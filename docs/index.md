# AI戦略実践プロジェクト

**「AIをツールとして便利に使う（AI as a tool）」から、「AI前提で事業戦略を構想し実装する（AI as a strategy）」へ。**

事業を動かす方々のための、AI戦略実装の「型」を月次で作る・動かす実践の場です。
各回ごとに事業戦略上の問いから出発し、AIを組み込んで戦略課題を解く仕組みを作り、
動かし、自社・自業界の競争前提がAIでどう書き換わるかまで踏み込みます。

主催：栗山実／株式会社アンテカニス
