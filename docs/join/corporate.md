# 法人の方へ

事業組織・社員にAI戦略の視座を持たせ、実行力を持ったAI・DX組織化を目指す——そうした
法人でのご利用にも対応しています。

## ご提供できること

- **複数名でのご利用** — 一定人数以上でのご参加
- **お見積書の発行・請求書払い** — 勤務先での購買・経費処理に対応
- **個別開催・テーマカスタマイズ**（費用別途） — 貴社の事業・業界文脈に合わせたテーマ設計
- **適格請求書（インボイス）対応** — インボイス番号付きのPDFを発行

## 本プロジェクトが提供する視座

「ツールとしてのAI活用」にとどまらず、**AI前提で事業戦略を構想・実装し、組織能力として
残す**という視座を、手を動かす実践を通じて組織に根づかせます。詳しくは
**[設計思想](../about/philosophy.md)** をご覧ください。
