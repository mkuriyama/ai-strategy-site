# 登録方法

参加・継続案内の受け取りは、登録フォームからのメール登録で行います。次回以降（月1回予定）の
参加案内、追加開催（サブ回）のご案内、今後の開催に関する継続的な情報を、順次メールでお送り
します。

以下のいずれのご意向でも、まずはご登録ください。

- 次回セッションへの参加を希望
- 日時が合わないが、録画視聴での参加を希望／別日程（サブ回）があれば参加希望
- 継続参加（月次）を検討
- 現時点では未定だが、本プロジェクトの継続的な案内を希望

!!! info "メール配信について"
    ご案内配信メールは **support@antecanis.com** からお送りします。プロモーションメール等に
    紛れて見落とされないよう、連絡先登録やメール振り分けでの対応をお勧めします。配信は
    いつでもご自身でキャンセルできます。ご登録後、開催案内とStripe決済のご案内を順次メールで
    お送りします（オンライン会場のZoom情報・当日資料は、お申し込み後にご案内します）。

## ご参加の流れ

1. 登録フォームからメール登録（無料）。次回以降の開催案内を順次メールで受け取る。
2. Stripe決済のご案内をメールで受け取る（参加コホートに応じて個別にご案内します）。
3. Stripeで参加のお申し込み（決済登録）。**申込当月回は無料**、翌月回から課金開始（いつでもご自身で解約可）。
4. お申し込み後に、オンライン会場（Zoom）情報・当日資料と、参加者コミュニティ（Circle）への
   ご招待を受け取る（会場リンク・資料は主に Circle 内でご案内します）。

参加費・課金のしくみは **[参加費と決済](index.md)** に図でまとめています。どなたに向けた
場かは **[背景と狙い](../about/index.md#想定参加者)** をご覧ください。
